# 🧩 Count Subsequences with Sum = K

## Problem Statement
Given an integer array `nums` and an integer `k`, return the total number of subsequences whose sum is equal to `k`.

---

## 🧪 Example

**Input:**
nums = [1, 3, 2, 1]  
k = 3  

**Output:**
3  

**Explanation:**
Valid subsequences are:
[1,2], [3], [2,1]

---

## 📌 Constraints
- 0 <= nums.length <= 10
- -10 <= nums[i] <= 10
- -100 <= k <= 100
