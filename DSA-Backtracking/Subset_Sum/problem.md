# 🧩 Subset Sum (Backtracking)

## Problem Statement
Given an integer array `nums` and an integer `k`, return all possible subsets whose sum is equal to `k`.

The solution set must not contain duplicate subsets.

---

## 🧪 Example

**Input:**
nums = [5, 9, 4]  
k = 9  

**Output:**
[[9], [5,4]]

---

## 📌 Constraints
- 0 <= nums.length <= 10
- -10 <= nums[i] <= 10
- -100 <= k <= 100
