# 🧩 Subsequence Sum Exists (Return Any One)

## Problem Statement
Given an integer array `nums` and an integer `k`, determine if there exists a subsequence whose sum is equal to `k`.

Return any one valid subsequence if it exists, otherwise return an empty list.

---

## 🧪 Example

**Input:**
nums = [5, 9, 4]  
k = 9  

**Output:**
[9]

(Another valid output: [5,4])

---

## 📌 Constraints
- 0 <= nums.length <= 10
- -10 <= nums[i] <= 10
- -100 <= k <= 100
